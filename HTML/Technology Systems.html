A technology system is the complete collection of components (hardware, software, peripherals, power supplies, communication links) making up a single computer installation. What this means is that a company or organisation will create a computer system designed to complete only one task or one set of tasks. At home your computer maybe set up to do a number of things, for example, some programming, graphics, video/sound editing and general social networking, shopping and gaming. An organisation wouldn't do this. Think about a bank ATM. This is a computer system set up to deal with financial transactions and nothing else. The same can be said for a ticket machine at a train station or a self service station at a supermarket.
